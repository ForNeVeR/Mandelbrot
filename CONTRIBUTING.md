Contributor Guide
=================

Prerequisites
-------------
To build Mandelbrot, you'll need a Rust toolchain.

## Windows
- Rust toolchain: 
  ```console
  $ choco install rust-ms
  ```
- CMake (for SDL2):
  ```console
  $ choco install cmake --version 3.31.7
  ```

Building
--------
Run the following shell command:
```
$ cargo build
```
